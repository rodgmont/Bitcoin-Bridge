# 🌉 Bitcoin Bridge

**Analyzing local commerce and purchasing power in El Salvador.**

Bitcoin Bridge is an analytical tool built for the **CUBO+ Build Sprint 2026**. Our system tracks retail price evolution from 2025 to 2026 to understand the real-world impact of Bitcoin adoption.
